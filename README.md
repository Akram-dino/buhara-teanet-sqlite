# Buhara TeaNet 🌿

AI-powered tea crop disease detection and advisory system.

## 📌 Overview
Buhara TeaNet is a web-based platform designed to help tea farmers detect diseases using image analysis and receive expert-reviewed recommendations.

## 🚀 Features
- Image-based disease detection (Roboflow AI)
- Role-based system (Worker, Reviewer, Admin)
- Real-time analytics dashboard
- PDF report generation
- Location capture with map preview
- Reviewer validation workflow

## 🛠 Tech Stack
- Django (Backend)
- Tailwind CSS (Frontend)
- Roboflow API (AI Model)
- SQLite (Development DB)
